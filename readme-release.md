# Hakatanimation version 0.1 alpha

## 完全無保証

このソフトウェアを使用したこと使用しなかったことによる精神的肉体的金銭的損害を含むとにかく一切に関して保証をいたしません。ソースコードは公開してるので動作に関してよくお読みになったのち、すべてご自身の責任でご使用ください。というか、自分でコンパイルしてください。

## 準備

* config.iniにニコニコのユーザー名とパスワードを入力してください。はい、平文保存です。

## 動作

* 起動すると、一時間に一度、ニコニコチャンネルでダウンロード出来るアニメをすべてダウンロードします。
* ダウンロードしたアニメは「video」フォルダに保存されます。
* 一度ダウンロードしたアニメは記録し二回目は取りに行きません。
* ダウンロードするたびにシェルが起動します。邪魔です。
* 最初のダウンロードはものごっつい時間がかかります。40GBくらいダウンロードします。

## タスクトレイアイコンの右クリックメニュー

### Execute now

一時間に一回の処理を今実行します。

### Restructure index

ダウンロードしたアニメを探索し、インデックスを再構築します。

### Exit

終了します。
